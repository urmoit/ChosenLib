Usage Guide

Learn how to use ChosenLib utilities effectively in your Fabric mods with practical examples and best practices.

üöÄ Quick Start

Get started with ChosenLib in just a few steps:

1. Import the Utilities

import com.chosen.lib.util.TextUtils;
import com.chosen.lib.util.GuiUtils; // Client-side only
import com.chosen.lib.util.ItemUtils;

2. Use Basic Functions

// Send a success message to chat
TextUtils.success("Operation completed successfully!");

// Check if an item stack is empty
boolean isEmpty = ItemUtils.isEmpty(player.getMainHandStack());

// Center a GUI element (client-side)
int centerX = GuiUtils.centerX(200);

üìù TextUtils Examples

TextUtils provides comprehensive text manipulation and formatting capabilities.

Basic Text Messages

// Send different types of messages
TextUtils.success("Task completed!");
TextUtils.error("Something went wrong!");
TextUtils.info("Here's some information");

// Create clickable links
TextUtils.clickable("Visit our website", "https://example.com");

Text Manipulation

// String manipulation
String capitalized = TextUtils.capitalize("hello world"); // "Hello world"
String repeated = TextUtils.repeat("abc", 3); // "abcabcabc"
String truncated = TextUtils.truncate("Very long text", 10); // "Very long..."
String reversed = TextUtils.reverse("hello"); // "olleh"

// Text validation
boolean isEmail = TextUtils.isValidEmail("user@example.com"); // true
boolean isAlpha = TextUtils.isAlphanumeric("abc123"); // true
String clean = TextUtils.removeSpecialChars("Hello@World!"); // "HelloWorld"

Visual Elements

// Create a progress bar
Text progressBar = TextUtils.progressBar(75, 100, 20);

// Center text in a specific width
String centered = TextUtils.centerText("Centered", 50);

// Create a text box
String textBox = TextUtils.createTextBox("Important message", 30);

Advanced Text Operations

// Check if text is palindrome
boolean isPal = TextUtils.isPalindrome("racecar"); // true

// Count words
int wordCount = TextUtils.wordCount("Hello world"); // 2

// Convert to title case
String title = TextUtils.toTitleCase("hello world"); // "Hello World"

üé® GuiUtils Examples (Client-side only)

GuiUtils provides GUI utilities for creating beautiful Minecraft interfaces.

Screen Positioning

// Center elements on screen
int centerX = GuiUtils.centerX(200); // Center 200px wide element
int centerY = GuiUtils.centerY(100); // Center 100px tall element

// Center within a specific range
int centered = GuiUtils.center(50, 200); // Center 50px in 200px range

Color Utilities

// Create colors
int red = GuiUtils.rgb(255, 0, 0);
int transparent = GuiUtils.rgba(255, 0, 0, 128);
int withAlpha = GuiUtils.withAlpha(red, 128);

// Convert HSV to RGB
int green = GuiUtils.hsvToRgb(120, 1.0f, 1.0f);

// Adjust brightness
int brighter = GuiUtils.adjustBrightness(red, 1.5f);

Drawing Shapes

// Draw a filled rectangle
GuiUtils.drawRect(context, 10, 10, 100, 50, red);

// Draw a bordered rectangle
GuiUtils.drawBorderedRect(context, 10, 10, 100, 50, fillColor, borderColor, 2);

// Draw a gradient
GuiUtils.drawGradient(context, 10, 10, 100, 50, startColor, endColor, true);

// Draw a texture
GuiUtils.drawTexture(context, texture, 10, 10, 0, 0, 32, 32, 64, 64);

UI Components

// Draw a progress bar
GuiUtils.drawProgressBar(context, 10, 10, 100, 20, 0.75, bgColor, progressColor);

// Check if mouse is over a rectangle
boolean isHovering = GuiUtils.isPointInRect(mouseX, mouseY, 10, 10, 100, 50);

üéí ItemUtils Examples

ItemUtils provides comprehensive item management and manipulation utilities.

Item Validation

// Check if item stacks are empty
boolean isEmpty = ItemUtils.isEmpty(stack);
boolean isNotEmpty = ItemUtils.isNotEmpty(stack);

// Compare items
boolean sameType = ItemUtils.isSameItem(stack1, stack2);
boolean exactMatch = ItemUtils.isExactMatch(stack1, stack2);

Item Information

// Get item information
String itemId = ItemUtils.getItemId(stack); // "minecraft:diamond_sword"
String displayName = ItemUtils.getDisplayName(stack); // "Diamond Sword"
int maxStack = ItemUtils.getMaxStackSize(item); // 64
boolean stackable = ItemUtils.isStackable(item); // true

Durability Utilities

// Check durability
int durability = ItemUtils.getDurability(stack);
int maxDurability = ItemUtils.getMaxDurability(stack);
boolean damaged = ItemUtils.isDamaged(stack);
double damagePercent = ItemUtils.getDamagePercentage(stack); // 0.0 to 1.0

Stack Manipulation

// Create a copy with different count
ItemStack newStack = ItemUtils.copyWithCount(stack, 5);

üéØ Real-World Examples

Here are some practical examples of how to use ChosenLib in your mods:

Custom GUI Screen

public class CustomScreen extends Screen {
    @Override
    public void render(DrawContext context, int mouseX, int mouseY, float delta) {
        super.render(context, mouseX, mouseY, delta);
        
        // Center the GUI
        int centerX = GuiUtils.centerX(200);
        int centerY = GuiUtils.centerY(100);
        
        // Draw background
        GuiUtils.drawRect(context, centerX, centerY, 200, 100, 
                         GuiUtils.rgba(0, 0, 0, 200));
        
        // Draw border
        GuiUtils.drawBorderedRect(context, centerX, centerY, 200, 100,
                                 GuiUtils.rgba(0, 0, 0, 200),
                                 GuiUtils.rgb(255, 255, 255), 2);
        
        // Check if mouse is hovering
        if (GuiUtils.isPointInRect(mouseX, mouseY, centerX, centerY, 200, 100)) {
            // Mouse is over the GUI
        }
    }
}

Item Processing System

public void processItem(ItemStack input, ItemStack output) {
    // Validate input
    if (ItemUtils.isEmpty(input)) {
        TextUtils.error("No input item provided!");
        return;
    }
    
    // Check if items are compatible
    if (!ItemUtils.isSameItem(input, output)) {
        TextUtils.error("Incompatible items!");
        return;
    }
    
    // Process the item
    if (ItemUtils.isDamaged(input)) {
        double damage = ItemUtils.getDamagePercentage(input);
        TextUtils.info("Processing damaged item (" + 
                      String.format("%.1f", damage * 100) + "% damaged)");
    }
    
    // Success message
    TextUtils.success("Item processed successfully!");
}

Command with Rich Text

public class CustomCommand {
    public static void execute(CommandContext context) {
        ServerCommandSource source = context.getSource();
        
        // Send formatted messages
        source.sendFeedback(() -> TextUtils.success("Command executed!"), false);
        
        // Send clickable links
        source.sendFeedback(() -> 
            TextUtils.clickable("Visit our website", "https://example.com"), false);
        
        // Send progress information
        Text progressBar = TextUtils.progressBar(75, 100, 20);
        source.sendFeedback(() -> progressBar, false);
    }
}

üí° Best Practices

Performance Tips

  • Cache frequently used values: Store color values and positions to avoid recalculating
  • Use appropriate utility methods: Choose the most efficient method for your use case
  • Validate inputs: Always check for null or empty values before processing

Code Organization

  • Import only what you need: Don't import unused utilities
  • Separate client and server code: Keep GuiUtils usage in client-side code only
  • Use meaningful variable names: Make your code readable and maintainable

Error Handling

  • Provide user feedback: Use TextUtils to inform users of errors and successes
  • Validate inputs: Check parameters before using utility methods
  • Handle edge cases: Consider what happens with null or invalid inputs
