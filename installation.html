Installation

How to add ChosenLib to your modding project.

Download

  • CurseForge
  • Modrinth (Coming Soon)
  • GitHub Releases

As a Mod

  1. Download the latest .jar and place it in your mods/ folder.
  2. Make sure Fabric Loader and Fabric API are installed.

As a Dependency (Gradle)

repositories {
    mavenCentral()
    maven { url = 'https://maven.fabricmc.net/' }
}
dependencies {
    modImplementation "com.chosen:chosenlib:1.2.1"
}
