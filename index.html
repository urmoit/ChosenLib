ğŸ’» Quick Start

Get up and running with ChosenLib in minutes:

1. Add as Dependency

dependencies {
    modImplementation "com.yourgroup:chosenlib:1.3.1"
}

2. Import Utilities

import com.chosen.lib.util.TextUtils;
import com.chosen.lib.util.GuiUtils; // Client-side only
import com.chosen.lib.util.ItemUtils;

3. Start Using

// Text utilities
TextUtils.success("Operation successful!");
TextUtils.clickable("Visit website", "https://example.com");

// GUI utilities (client-side)
int centerX = GuiUtils.centerX(200);
GuiUtils.drawRect(context, x, y, width, height, color);

// Item utilities
boolean isEmpty = ItemUtils.isEmpty(stack);
String itemId = ItemUtils.getItemId(stack);
