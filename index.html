ChosenLib v1.3.1

A comprehensive utility library for Fabric that streamlines common modding tasks. Build better mods faster with our collection of GUI, text, and item utilities.

🚀 Get Started 📚 API Reference 💻 GitHub
🎨

GuiUtils

Client-side GUI utilities for creating beautiful Minecraft interfaces. Screen positioning, color manipulation, drawing helpers, and more.

  • Screen positioning helpers
  • Color utilities and gradients
  • Drawing primitives
  • Progress bars and hit testing
📝

TextUtils

Enhanced text manipulation and formatting utilities. From basic string operations to advanced visual elements and clickable links.

  • Text manipulation and validation
  • Visual elements and formatting
  • Clickable chat links
  • Progress bars and text boxes
🎒

ItemUtils

Comprehensive item management and manipulation utilities. Item validation, comparison, information retrieval, and durability handling.

  • Item validation and comparison
  • Item information retrieval
  • Durability utilities
  • Stack manipulation

💻 Quick Start

Get up and running with ChosenLib in minutes:

1. Add as Dependency

dependencies {
    modImplementation "com.yourgroup:chosenlib:1.3.1"
}

2. Import Utilities

import com.chosen.lib.util.TextUtils;
import com.chosen.lib.util.GuiUtils; // Client-side only
import com.chosen.lib.util.ItemUtils;

3. Start Using

// Text utilities
TextUtils.success("Operation successful!");
TextUtils.clickable("Visit website", "https://example.com");

// GUI utilities (client-side)
int centerX = GuiUtils.centerX(200);
GuiUtils.drawRect(context, x, y, width, height, color);

// Item utilities
boolean isEmpty = ItemUtils.isEmpty(stack);
String itemId = ItemUtils.getItemId(stack);
