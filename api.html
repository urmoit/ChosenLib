API Reference

Classes

  • ChosenLib - Main entrypoint and general utilities
  • TextUtils - Text formatting and helpers
  • Config - Configuration system

Key Methods

ChosenLib

  • randomInt(int min, int max) - Get a random integer in a range
  • clamp(int value, int min, int max) - Clamp a value between min and max
  • lerp(double a, double b, double t) - Linear interpolation
  • isBetween(int value, int min, int max) - Check if a value is within a range
  • safeEquals(Object a, Object b) - Null-safe equality check
  • debugLog(String message) - Log debug messages

TextUtils

  • success(String message) - Green success text
  • error(String message) - Red error text
  • capitalize(String input) - Capitalize first letter
  • repeat(String str, int times) - Repeat a string
  • isNullOrEmpty(String str) - Check if null or empty
  • join(String separator, String... elements) - Join strings
  • combine(Text... texts) - Combine Text objects
  • stripFormatting(Text text) - Get plain string
  • format(String message, Formatting... formattings) - Format with multiple options
  • clickable(String label, String url) - Create clickable chat text

Config

  • See Config.java for available fields and usage.
