API Reference

Comprehensive documentation for all ChosenLib utility classes and methods.

üìù TextUtils

Enhanced text manipulation and formatting utilities for Minecraft modding.

Basic Text Operations

Method Description Example
success(String message) Creates a green success message TextUtils.success("Operation completed!")
error(String message) Creates a red error message TextUtils.error("Something went wrong!")
info(String message) Creates a blue info message TextUtils.info("Information here")
capitalize(String input) Capitalizes first letter TextUtils.capitalize("hello") // "Hello"
repeat(String str, int times) Repeats a string TextUtils.repeat("abc", 3) // "abcabcabc"

Advanced Text Manipulation

Method Description Example
truncate(String str, int maxLength) Truncates string with ellipsis TextUtils.truncate("Very long text", 10) // "Very long..."
toTitleCase(String str) Converts to title case TextUtils.toTitleCase("hello world") // "Hello World"
reverse(String str) Reverses a string TextUtils.reverse("hello") // "olleh"
isPalindrome(String str) Checks if string is palindrome TextUtils.isPalindrome("racecar") // true
wordCount(String str) Counts words in string TextUtils.wordCount("Hello world") // 2

Text Validation

Method Description Example
isNullOrEmpty(String str) Checks if null or empty TextUtils.isNullOrEmpty("") // true
isAlphanumeric(String str) Checks if alphanumeric TextUtils.isAlphanumeric("abc123") // true
isValidEmail(String email) Validates email format TextUtils.isValidEmail("test@example.com") // true
removeSpecialChars(String str) Removes special characters TextUtils.removeSpecialChars("Hello@World!") // "HelloWorld"

Visual Elements

Method Description Example
progressBar(int current, int max, int length) Creates text-based progress bar TextUtils.progressBar(75, 100, 20)
centerText(String text, int width) Centers text in given width TextUtils.centerText("Centered", 50)
createTextBox(String text, int width) Creates bordered text box TextUtils.createTextBox("Text in box", 30)
clickable(String message, String url) Creates clickable chat link TextUtils.clickable("Visit", "https://example.com")

üé® GuiUtils (Client-side only)

GUI utilities for creating beautiful Minecraft interfaces. Note: This class is client-side only.

Screen Positioning

Method Description Example
center(int value, int range) Centers value within range GuiUtils.center(100, 200) // 50
centerX(int width) Centers horizontally on screen GuiUtils.centerX(200)
centerY(int height) Centers vertically on screen GuiUtils.centerY(100)

Color Utilities

Method Description Example
rgba(int r, int g, int b, int a) Creates RGBA color GuiUtils.rgba(255, 0, 0, 255) // Red
rgb(int r, int g, int b) Creates RGB color (opaque) GuiUtils.rgb(255, 0, 0) // Red
withAlpha(int rgb, int alpha) Adds alpha to RGB color GuiUtils.withAlpha(red, 128)
hsvToRgb(float h, float s, float v) Converts HSV to RGB GuiUtils.hsvToRgb(120, 1.0f, 1.0f) // Green
adjustBrightness(int color, float brightness) Adjusts color brightness GuiUtils.adjustBrightness(red, 1.5f)

Drawing Methods

Method Description Example
drawRect(DrawContext context, int x, int y, int width, int height, int color) Draws filled rectangle GuiUtils.drawRect(context, 10, 10, 100, 50, red)
drawBorderedRect(DrawContext context, int x, int y, int width, int height, int fillColor, int borderColor, int borderWidth) Draws bordered rectangle GuiUtils.drawBorderedRect(context, 10, 10, 100, 50, fill, border, 2)
drawGradient(DrawContext context, int x, int y, int width, int height, int startColor, int endColor, boolean horizontal) Draws gradient rectangle GuiUtils.drawGradient(context, 10, 10, 100, 50, start, end, true)
drawTexture(DrawContext context, Identifier texture, int x, int y, int u, int v, int width, int height, int textureWidth, int textureHeight) Draws texture GuiUtils.drawTexture(context, texture, 10, 10, 0, 0, 32, 32, 64, 64)
drawProgressBar(DrawContext context, int x, int y, int width, int height, double progress, int backgroundColor, int progressColor) Draws progress bar GuiUtils.drawProgressBar(context, 10, 10, 100, 20, 0.75, bg, progress)

Utility Methods

Method Description Example
isPointInRect(int x, int y, int rectX, int rectY, int rectWidth, int rectHeight) Checks if point is in rectangle GuiUtils.isPointInRect(mouseX, mouseY, 10, 10, 100, 50)

üéí ItemUtils

Item management and manipulation utilities for Minecraft items and stacks.

Item Validation

Method Description Example
isEmpty(ItemStack stack) Checks if stack is empty ItemUtils.isEmpty(stack) // true/false
isNotEmpty(ItemStack stack) Checks if stack is not empty ItemUtils.isNotEmpty(stack) // true/false
isSameItem(ItemStack stack1, ItemStack stack2) Compares item types ItemUtils.isSameItem(stack1, stack2) // true/false
isExactMatch(ItemStack stack1, ItemStack stack2) Compares everything including NBT ItemUtils.isExactMatch(stack1, stack2) // true/false

Item Information

Method Description Example
getItemId(Item item) Gets item registry ID ItemUtils.getItemId(item) // "minecraft:diamond_sword"
getItemId(ItemStack stack) Gets item registry ID from stack ItemUtils.getItemId(stack) // "minecraft:diamond_sword"
getDisplayName(ItemStack stack) Gets display name ItemUtils.getDisplayName(stack) // "Diamond Sword"
getMaxStackSize(Item item) Gets maximum stack size ItemUtils.getMaxStackSize(item) // 64
isStackable(Item item) Checks if item is stackable ItemUtils.isStackable(item) // true/false

Durability Utilities

Method Description Example
getDurability(ItemStack stack) Gets current durability ItemUtils.getDurability(stack) // 1500
getMaxDurability(ItemStack stack) Gets maximum durability ItemUtils.getMaxDurability(stack) // 2501
isDamaged(ItemStack stack) Checks if item is damaged ItemUtils.isDamaged(stack) // true/false
getDamagePercentage(ItemStack stack) Gets damage percentage (0.0 to 1.0) ItemUtils.getDamagePercentage(stack) // 0.4

Stack Manipulation

Method Description Example
copyWithCount(ItemStack stack, int count) Creates copy with different count ItemUtils.copyWithCount(stack, 5)
